Estrutura Básica do HTML

Voltar

A estrutura básica do HTML consiste em três tags: <html>, <head> e <body>.

<html>: É a tag que inicializa e finaliza a página, ou seja, tudo que estiver dentro dela faz parte da página.

<head>: Signifca cabeça, ou seja, o cabeçãrio da página nessa tag ficam armazenadas todas as informações antes que tudo seja exibido para o usuário.

<body>: Significa corpo, ou seja, o corpo da página, tudo que aparecer dentro dessa tag irá aparecer para o usuário

Exemplo da estrutura básica do HTML

<html>

<head></head>

<body></body>

</html>

Atributo das Tags

Voltar

Atributos são as propriedades que a tag pode ter, existem atributos que são mais globais, que todas as tags vão ter e outros atributos que são mais específicos de cada tag.

Atributos globais:

São atributos que podem ser usados em qualquer tag.

id:

Na tag “strong” por exemplo podemos utilizar o atributo “id”. É um atributo que trabalha como identificador de elementos, ou seja, serve para você identificar através do java script qual elemento você deseja alterar.

Ex.:

<strong id="titulo"<Meu primeiro <i>HTML!<i></strong>

style:

Na tag “strong” podemos utilizar o atributo “style”, é um atributo quem tem a finalidade de colocar comandos do CSS direto no elemento. a título de exemplo podemos utilizar o comando "color".

Ex:

<strong> style="color: ☐blue;"<Meu primeiro <i>HTML<i></strong>

class:

É um atributo votado para o CSS que tem como finalidade padronizar qualquer parte do seu texto no HTML, por exemplo, colocando uma palavra, parte do texto ou elemento em uma única cor.

Ex.:

<strong class="título principal"<Meu primeiro <i>HTML<i></strong>

Atributos mais específicos de cada tag:

<input />

A tag <input /> tem como finalidade configurar um campo texto, local para digitar na página, muito utilizado para criação de formulários de cadastro em sites. A mesma possui um atriguto específico chamado "type".

type

Configura o campo da tag <input /> para receber textos, numeros e cor através dos comandos “text”, "number" e "color" entre outros.

text

Cria campo de tipo texto para ser digitado pelo usuário na página html.

Ex.:

<input type="text" />

number

Cria campo de tipo numero para ser digitado pelo usuário na página html.

Ex.:

<input type="number" />

color

Cria campo de tipo cor para ser escolhido pelo usuário na página html.

Ex.:

<input type="color" />


<img />

Basicamente a tag coloca uma imagem na página utilizando os atributos específicos “src" e "width" entre outros.

src

Serve para fornecer o caminho da imagem escolhida, esteja ela dentro do computador ou em um link de página web

Exemplo de linha de comando carregando imagem atraves de arquivo localizado no computador:

<img src="carro.png" />

Exemplo de linha de comando carregando imagem atraves de link da internet:

<img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSbIgH0LiG93oflP5F_3iHc6aqWyDlElEHJtZLweWr3eQ&s" />

width

Através do atributo "width" podemos ajustar a largura da imagem adicionada a página.

Ex:

<img width="150" src="carro.png" />


Textos

Voltar

Textos basicamente é o que o usuário vai ler, ele vai entrar no site e vai ler textos ou seja, tipografia. Existem várias tags para ser usadas nos textos.

<h>

Significa head ou seja, cabeçário ou título. A tag “h” vem acompanhada de níveis poque existem títulos e subtítulos e assim por diante subsequentemente temos as tags “h1, h2, h3, h4, h5 e h6”, sendo h1 o título e os demais de h2 a h6 subtítulos.

Ex.:

<html>

<head>

<title<Mudei de título</title>

</head>

<body>

<h1>DIO.me</h1>

<h2>Implemente seu futuro global agora!</h2>

<h3>Estudo com os melhores especialistas!</h3>

<h4>Material digital totalmente gratuíto!</h4>

<h5>Aproveite os os melhores bootcamps!</h5>

<h6>Aumente a sua chance de engressar no mercado de trabalho!</h6>

</body>

</html>


<p>

É a tag de parágrafo. Onde fica o corpo do texto da sua página html. A cada parágrafo é necessário abir uma tag <p> nova.

Ex.:

<html>

<head>

<title<Mudei de título</title>

</head>

<body>

<h1>DIO.me</h1>

<h2>Implemente seu futuro global agora!</h2>

<p>Domine as teclonogias dominadas pelas empresas.</p>

</body>

</html>

<blockquote>

Essa tag tem a função de configurar citações, ou seja, uma frase de outra pessoa ou uma observação que de forma você quer chamar a atenção para depois explicar mais sobre aquela palavra. O html entende isso e exibe a citação ou observação dando uma margem diferente jogando o texto um pouco para a direita desprendendo do texto e depois você pode formatar no CSS do jeito que quiser colocando um marcador em forma de bola por exemplo entre outros.

Ex.:

<html>

<head>

<title<Mudei de título</title>

</head>

<body>

<h1>DIO.me</h1>

<h2>Implemente seu futuro global agora!</h2>

<p>Domine as teclonogias dominadas pelas empresas.</p>

<blockquote>O jeito Dio. para você ganhar destaque mais rápido no mercado.</blockquote>

</body>

</html>

font

É utilizada para configurar o tipo, cor e tamanho das letras do texto de uma página html.

    Cor

    É inserida da seguinte forma:

    Ex.:

    <font color="Red">texto</font>

    Tipo

    Que é definido das seguintes formas:

    • Com uma única fonte

      • Ex.:

      <font face="serif">Texto</font>


    • Com combinação de tipos de fontes

      • Ex.:

      <font face="Verdana,Geneva,Arial,Helvetica,sans-serif">Texto</font>

    OBS: O navegador procurará primeiro no computador do leitor o arquivo de fonte Verdana, caso ele encontre ele exibirá o texto com fontes do tipo Verdana. Se não encontrar procurará o arquivo de fonte Geneva, para exibir o texto e assim sucessivamente.

    Tamanho

    Que é definido das seguintes formas:

    • Relativo

      • Que variam dos tamanhos de +1 a +6

      Ex.:

      <font size="+1">Texto</font>


    • Absoluto

      • Que variam de tamanho de 1 a 7.

      Ex.:

      <font size="2">Texto</font>


    • Relativo

      • Que variam dos tamanhos de -1 a -6.

      Ex.:

      <font size="-5">Texto</font>


    Combinação dos três atributos

    Ex.:

    <font face="Arial,Helvertica,sans-serif" size="+1" color="Maroon">Texto;</font>




    OBS: Caso deseje em negrito.

    Ex.:

    <font face="Arial,Helvertica,sans-serif" size="+1" color="Maroon"><b>Texto</b></font>


    del

    É utilizado para indicar que um texto foi suprimido, apesar de, aparecer riscado, para que o leitor tenha ciência da ocorrência.

    Pode-se designar o atributo “CITE=” dentro da tag para informar a data de supressão do texto.

    Ex.:

    <del "cite=15/06/2011"></del>

    A critério do editor, após o texto contido dentro da tag “del”, pode-se colocar uma informação recente usando-se a tag “INS” em substituição ao texto suprimido, que aparecerá destacado.

    Ex.:

    <p>A Equipe A <del>perdeu</del> <ins>ganhou</ins> a partida realizada no dia 15/06/2021 para a Equipe B.

    Visualização no navegador:

    A Equipe A perdeu ganhou a partida realizada no dia 15/06/2021 para a Equipe B.

    <abbr>

    A tag <abbr> é utilizada para indicar abreviaturas ou siglas.

    Marcando-se abreviaturas que você pode dar informações uteis para os navegadores, corretores ortográficos, sistemas de tradução e indexadores.

    Pdemos definir abreviações com o elemento abbr. Por padrão, esse elemento é exibido como elemento de linha. O atribudo title desse elemento é utilizado para definir um tootip.

    O atributo title pode ser usado na tag <abbr> para mostrar a versão completa da abreviatura/sigla, quando você passa mouse sobre o elemento <abbr>.

    Ex.:

    <abbr title="Sport Clube Recife">S.C.R</abbr> é o melhor time do mundo.


    Listas Ordenadas e Não Ordenadas

    Voltar

    As listas geralmente são utilizadas dentro de um texto para construção de um sumário por exemplo porém CSS é muito utilizada para construção de menus.

    Listas Ordenadas

    Para iniciar uma lista ordenada é utilizada a tag “<ol>” juntamente com os itens da lista que são adicionados pela tag filha “<li>”. Quando se aplica uma lista ordenada o navegador vai entender que a lista será exibida por números ordenadamente. Também é possível colocar uma lista dentro de outa ou dentro de outro item.

    Ex.:

    <ol>

            <li>item1</li>

            <li>item2</li>

            <li>item5</li>

                                   <ol>

                                           <li>item1</li>

                                           <li>item2</li>

                                           <li>item5</li>

                                   <ol>

    </ol>


    Listas Não Ordenadas

    Para iniciar uma lista não ordenada é utilizada a tag “<ul”> juntamente com os itens da lista que são adicionados pela tag filha “<li”> que são visualizadas como marcadores.

    Ex.:

    <ul>

            <li>item1</li>

            <li>item2</li>

            <li>item5</li>

    </ul>


    Links

    Voltar

    Dentro de um site nos clicamos em um link vamos para outra página, quando isso acontece estamos navegando na internet, ou seja, nós vamos navegando através de uma ligação de uma pagina para outra e essa ligação é chamada de link.
    Para criação de um link iremos usar uma tag específica chamada “ancora” representada pela letra “a” utilizando o atributo “href”: "<a href="">".

    Ex.:

    <h1>Meu primeiro site</h1>

    <ul>

            <li>

                   <a href="http://dio.me"</a>

            </li>

    </ul>


    Também podemos utilizar o atributo “target” para navegar de uma página para outra através de um link configurando para abrir na mesma aba através do complemento “_self” que é o comportamento padrão de um link.

    Ex.:

    <h1>Meu primeiro site</h1>

    <ul>

            <li>

                   <a href="http://dio.me" target="_self">Dio.me</a>

            </li>

    </ul>


    OBS: Mesmo sendo utilizado o recurso “target” juntamento com o complemento “_self” podemos abrir uma página através de um link de outra página em outra aba utilizando o um recurso que o próprio navegador oferece segurando a tecla “CRTL” ao clicar em cima do link. No caso de segurarmos a tecla “SHIFT” o site é aberto através de uma nova janela do navegador.

    Também podemos utilizar o atributo “target” navegar de uma página através de um link página configurando para abrir em outra aba através do complemento “_blank”.

    Ex.:

    <h1>Meu primeiro site</h1>

    <ul>

            <li>

                   <a href="http://dio.me" target="_blank">Dio.me</a>

            </li>

    </ul>


    Também podemos configurar para que apareça um balão com um letreiro de informação utilizando o atributo “title”. Isso ajuda a navegabilidade melhorando a experiência do usuário no site.

    Ex.:

    <h1>Meu primeiro site</h1>

    <ul>

            <li>

                   <a href="http://dio.me" title="Clique aqui e confira!" target="_blank">Dio.me</a>

            </li>

    </ul>


    Podemos criar um novo site lincando com o site principal para voltar. Agraves da tag “a”

    Ex.:

    Página principal

    <html>

                <head>

                             <title>Página principal</title>

                </head>

                <body>

                             <h1>Meu primeiro site</h1>

                             <ul>

                                     <li>

                                            <a href="about.html" title="Clique aqui e confira!" target="_self">Sobre a minha empresa</a>

                                     </li>

                             </ul>

                </body>

    </html>


    Sobre a minha empresa

    <html>

                <head>

                             <title>Sobre minha empresa</title>

                </head>

                <body>

                             <h1>Sobre meu site</h1>

                             <a href="index.html" >Voltar</a>

                </body>

    </html>


    OBS: É recomendado que quando o link for o seu site dar preferencia para abrir na mesma aba, porém quando o link for para outro site que não seja de sua propriedade configurar para abrir em outra aba visando a importância de que o usuário não sai a do seu site.
